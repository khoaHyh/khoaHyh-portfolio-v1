About

This is the final project for the Udemy Course: 'Zero To Mastery' by Andrei Neagoie. The challenge was to create an app that detects faces upon the submission of a link to an image. The project uses the Clarifai Machine Learning Image Recognition API to detect faces. There is also login & register functionality. Each account that signs up has a score that increments for each time the user submits a link that successfully detects a face. That score is stored in a database on a Heroku server so that when the user logs back in, your account score is still there.
